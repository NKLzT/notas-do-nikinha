O que é o Spring Boot

Primeiramente saber oque é o spring boot e como ele funciona é enssencial para ultilizá lo?
Defenitivamente não, porem saber oque é e como ele funciona é importante por isso vamos brevente passar por esse assunto.

O Spring Boot veio pra solucionar um problema que era muito comum antigamente a necessidade de configuração e conhecimentos relacionados a Aplicações Web entre outras,
o spring boot trouxe com sigo uma solução para os "startups" de projetos podendo já funcionar com pouca ou nenhuma configuração.
Dentro do spring boot existem disversas dependencias que podemos adiciona-las ou retira-las baseado na sua necessidade, como por exemplo o spring-boot-starter-web, que traz com sigo diversas coisas para a construção de API's ou Sites.
